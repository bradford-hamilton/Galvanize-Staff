# Galvanize Staff

An app for tracking Galvanize staff and their associate cohorts

Link to live site => https://galvanize-staff.herokuapp.com/

## Installation

* Set up a postgres database
* Create a `.env` file, use `.env.example` as a reference
* `npm install knex -g`
* `npm install`
* `knex migrate:latest`
* `knex seed:run`
* `npm run dev-start`
